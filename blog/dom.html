the DOM and how it relates to HTML and CSS

To build a website, we need three languages, HTML, CSS and Javascript. Those 3 languages interact with each other and have different roles.

HTML (Hypertext Markup Language) is the language we use to create the base structure of a website. It should include the element, a with and <meta> elements, and the <body> with headings, paragraphs, images etc. But this content looks pretty boring on its own.<br> <br> CSS (Cascading Stylesheet) is the coding language used to style the HTML content, and make it less boring! It can target HTML semantic elements as well as classes and ids to change their visual appearance such as the background colour, font size, position etc, as well as add some basic interactions (mousehover, focussed etc).<br> <br> Javascript is a coding language used to add functionality to HTML, as well as a lot more. But Javascript needs the DOM to interact with HTML content.<br> <br> The DOM (Document Object Model) is not a programming language, it is the representation of structures and content of web documents. <br> <br> The DOM represents the document content as nodes and objects.
